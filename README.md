## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/Faiz2172/Dynamic_Portfolio_App.git
cd Dynamic_Portfolio_App
```

### 2. Install dependencies

```bash
cd frontend
npm install
```

### 3. Run the dev server

```bash
npm run dev
```

### 4. Build & deploy

```bash
npm run build
```
